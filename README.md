# Tingog Mo, Aksyon Ko - Community Suggestion Platform

## Setup (XAMPP)

1. Start XAMPP Apache and MySQL.
2. Open phpMyAdmin (http://localhost/phpmyadmin).
3. Create DB `tingog` or import `database.sql`.
4. Access site: http://localhost/`leza sys`/index.php
   - Note: Space in path? Rename dir or use http://localhost/leza%20sys/index.php

## Admin
- Go to admin.php
- Password: `admin123` (change in config.php)

## Flow Test
1. Submit suggestion from submit_form.php
2. Admin: login, edit status/response
3. Public: view updates.php (non-pending)

## Features
- Anonymous submit
- Categories
- Status tracking
- Professional responsive design (Roboto, blue theme, navbar)

Enjoy the community platform!
